# Cxyier Programming Language

A new compiled programming language implemented in C++23.

## Building

```bash
mkdir build
cd build
cmake ..
make
```

## Project Structure

- `src/` - Source code
- `include/` - Header files
- `tests/` - Test files
- `examples/` - Example programs (when we have some!)

## Requirements

- C++23 compatible compiler (GCC 12+, Clang 15+, or MSVC 19.30+)
- CMake 3.20+
